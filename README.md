## 实现点击加载更多评论的功能
1.为加载更多按钮，绑定点击事件，在事件中，请求下一页数据
2.点击加载更多，让 pageIndex++ ，然后重新调用 this.getComments() 方法重新获取最新一页数据
3.为了防止新数据覆盖老数据的情况，我们在点击加载更多的时候，每当获取到新数据，应该让老数据调用数组的 concat 方法，拼接上新数组
